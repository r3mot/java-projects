<h1 align="center">How to Play</h1>

- Select the size of your grid

- Select the number of each ship type (or let the program decide)

- Place your ships by selecting a ship from the drop down menu

- NOTE: Primary mouse button will place horizontally, Secondary mouse button will place vertically

- Play against the computer!
